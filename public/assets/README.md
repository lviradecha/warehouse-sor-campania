# Assets

Questa cartella contiene le risorse grafiche del sistema.

## Logo CRI

Aggiungi qui il logo della Croce Rossa Italiana:
- Nome file: `logo-cri.png`
- Dimensioni consigliate: 200x200px o superiore
- Formato: PNG con sfondo trasparente

Il logo verrà visualizzato nell'header del sistema.

## Favicon

Puoi anche aggiungere una favicon personalizzata:
- Nome file: `favicon.ico`
- Dimensioni: 32x32px o 16x16px

Se non hai il logo, il sistema funzionerà comunque, semplicemente non verrà visualizzata l'immagine.
